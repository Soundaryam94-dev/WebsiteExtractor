Website URL to Single Page Application ZIP Generator

Complete Requirements Document

---

1. Project Overview

Project Name

Website Extractor(Single Page Aplication)

Goal

The User pastes a website URL.

System automatically:

The Website gets analyzed.

The Content gets extracted.

The Images gets downloaded.

The Color palette gets detected.

The Typography gets detected.

The Single Page Application gets generated.

The ZIP file gets created

Final:

website-assets.zip

It should be downloaded by user.

---

1. Main Workflow

User Input URL
↓
Frontend API Request
↓
Backend Receives URL
↓
Playwright Opens Website
↓
Website Fully Render
↓
Extract:

- Images
- Colors
- Typography
- Content
- HTML
↓
Generate Design System
↓
Create Single Page Application Structure
↓
Generate ZIP File
↓
Send ZIP to Frontend
↓
User Downloads ZIP
